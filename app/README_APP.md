## Web UI
### Install Dash
```
pip install dash==0.39.0  # The core dash backend
pip install dash-renderer==0.13.0  # The dash front-end
pip install dash-html-components==0.12.0  # HTML components
pip install dash-core-components==0.30.0  # Supercharged components
pip install plotly --upgrade  # Latest Plotly graphing library
```
